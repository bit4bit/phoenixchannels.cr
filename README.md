# phoenixchannels

Client of phoenixframework channel

## Installation

1. Add the dependency to your `shard.yml`:

   ```yaml
   dependencies:
     phoenixchannels:
       github: bit4bit/phoenixchannels.cr
   ```

2. Run `shards install`

## Usage

```crystal
require "phoenixchannels"
```
